Fiver for Game Boy

This is a game like Wordle. The answers list, however, has been replaced with one generated from English usage frequency data (with some words removed for family-friendliness and other reasons).

Controls

  • Arrows: Game Boy dpad
  • Z: Game Boy A button (select letter)
  • X: Game Boy B button (backspace)
  • Enter: Game Boy Start button (enter word)
